# 🏥 병원 공간기획 3D 재배치 스튜디오

상급종합병원 층별 공간을 3D로 배치·재배치하고, 면적 증감·법령 검토·동선·AI 자동배치까지 지원하는 웹 앱입니다.

## 바로 사용 (GitHub Pages)
👉 **https://paul-kim1014.github.io/hospital-3d-space-planner/**

설치 없이 브라우저에서 바로 열립니다. PC 권장(크롬/엣지/사파리).

## 주요 기능
- 3D 입체 배치 · 용도별 색상 분류 · 레고식 이동/추가/삭제
- 층 간격 조절 · 층 단독 편집(포커스) · 순서 재배치 · 수평 펼쳐보기
- PDF/DXF 도면 업로드(밑그림) · **DXF 실 명칭·면적 자동 인식**
- 좌측 카테고리 분류 · 실 외곽선(폴리곤) 그리기
- 환자/물류 **동선 그리기**(색상·굵기·지우개) · 선택 도형 **명칭 표시**
- 면적 증감표 · 안 비교 · 보고서(인쇄/PDF) · 캡처
- 관련 법령(국가법령정보센터) 바로가기 · 채팅형 **AI 자동배치**
- 작업 히스토리 · 실행취소(Ctrl+Z)

## 데이터 저장
- 작업은 **각자 사용하는 브라우저에 자동 저장**됩니다(기기별).
- 다른 기기로 옮기려면 앱 안의 `⬇JSON` 내보내기 → 다른 기기에서 `⬆불러오기` 하세요.

> 여러 사용자의 데이터를 서버 한 곳에 모아 기기 간 자동 동기화하려면 별도 서버(예: Railway) 배포가 필요합니다.
